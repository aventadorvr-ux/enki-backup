## 6. Mobile Responsive Considerations

### Mobile Navigation Pattern
```
+--------------------------------+
| [=]  [Logo]    [Search] [User] |
+--------------------------------+
```

- Hamburger menu for sidebar navigation
- Bottom tab bar for primary actions
- Full-width cards in single column
- Floating action button for key actions
- Swipe gestures for property galleries

### Mobile Property Card

## 7. Component Specifications

### Property Card
- Width: 320px (desktop), 100% (mobile)
- Image height: 200px
- Border radius: 12px
- Shadow: 0 2px 8px rgba(0,0,0,0.1)
- Hover: translateY(-4px), shadow increase

### KPI Card
- Width: Flexible (grid)
- Padding: 24px
- Border radius: 12px
- Background: White or subtle gradient
- Number: 36px, bold
- Label: 14px, muted color

### AI Chat Bubble
- Max width: 80%
- Padding: 16px
- Border radius: 16px (user), 16px 16px 4px 16px (AI)
- Background: Primary color (user), Gray-100 (AI)
