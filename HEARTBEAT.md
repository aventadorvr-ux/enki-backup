# HEARTBEAT.md - Periodic Operational Checks

## Schedule: Run checks every ~4 hours during day (8am-10pm local)

### Morning Checklist (first heartbeat of day)
- [ ] Review memory/today notes from yesterday
- [ ] Scan for trending topics in income niches (if content path chosen)
- [ ] Check active project status / any blockers
- [ ] Note 1 priority action for today

### Afternoon Checklist
- [ ] Progress on active income projects
- [ ] Any new opportunities surfaced?
- [ ] System health check (disk, updates pending)

### Evening Checklist (last heartbeat before 10pm) — ✅ 2026-02-28
- [x] Log today's wins to memory/2026-02-28.md
- [x] Update MEMORY.md with key learnings
- [x] Prep tomorrow's priority

### Evening Checklist (last heartbeat before 10pm)
- [ ] Log today's wins to memory/YYYY-MM-DD.md
- [ ] Update MEMORY.md with key learnings
- [ ] Prep tomorrow's priority
- [ ] Git backup to GitHub (auto-every 6h via cron, or manual if needed)
