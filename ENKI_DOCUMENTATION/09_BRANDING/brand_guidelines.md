# ENKI AGENTIC Brand Guidelines
## Version 1.0

## 1. Brand Identity
**Name:** ENKI AGENTIC  
**Tagline:** We Bring The ME  
**Namesake:** ENKI - Sumerian god of wisdom, creation, intelligence

## 2. Color Palette
- **Primary:** ENKI Blue (#0066FF), ENKI Gold (#FFD700)
- **Secondary:** Deep Navy (#0A1929), Electric Cyan (#00D9FF), Alert Red (#FF3366)
- **Neutrals:** Pure White, Soft Gray, Medium Gray, Dark Gray

## 3. Typography
- **Primary:** Inter (400, 500, 600, 700)
- **Technical:** JetBrains Mono
- **Display:** Space Grotesk

## 4. Logo Concepts
1. **Digital Caduceus** - Modern twin serpents (primary)
2. **ME Symbol** - 8-pointed star (icon)
3. **Water of Wisdom** - Flowing water with binary

## 5. Voice & Tone
Confident but not arrogant. Technical but accessible. Visionary but grounded.

**Use:** Agents, automation, intelligence, robust, deploy, dominate  
**Avoid:** AI magic, robots taking over, toy, demo

## 6. Frontend Styling
- **Framework:** Tailwind CSS + shadcn/ui
- **Theme:** ENKI Dark (dark mode first)
- **Components:** Agent cards, status indicators, data displays
- **Animations:** Pulse, gradient shift, alert flash
